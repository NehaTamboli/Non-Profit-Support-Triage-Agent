# Non-Profit-Support-Triage-Agent
Non-Profit Support Triage Agent is an AI-powered communication automation system designed to help non-profit organizations efficiently manage incoming messages such as donation inquiries, volunteer requests, complaints, and general queries. The system uses Natural Language Processing (NLP), Large Language Models, LangChain framework, and a Streamlit-based dashboard to automatically classify messages, extract key information through Named Entity Recognition (NER), generate contextual response drafts, and provide analytics insights. It also uses a vector database (ChromaDB) for semantic storage and retrieval, helping improve response accuracy and communication efficiency while reducing manual workload.
